# Lure Designer(プロトタイプ)

ブラウザ上でオリジナルルアーをパラメトリックにデザインし、3Dプリント用STLを出力できるWebアプリのプロトタイプです。

**▶ 試す: https://notthi.github.io/luredesign/**

## 機能

- テンプレート(アジング〜シーバス、30〜150mm)から選んでスライダーで形状調整
- リアルタイム3Dプレビュー(スマホのタッチ操作対応)
- 下部ソリッド+背部中空の内部構造を自動生成(FDM印刷向け)
- 浮力・重量の目安計算(PETG想定、淡水/海水切替)
- アクション傾向・潜行深度の簡易表示(経験則ベース)
- バイナリSTLのダウンロード

## 使い方

ページを開くだけで動きます(ビルド不要・データはすべてブラウザ内で処理)。
STLはBambu Studio等のスライサーにそのまま読み込めます。
